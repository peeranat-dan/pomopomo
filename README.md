# PomoPomo
URL: [https://pomo-pomo.netlify.app/home](https://pomo-pomo.netlify.app/home)
- **Pomodoro Timer**
- Coded with **ReactJS + Vite**
- Styling with **Tailwind CSS**
- Create a backend using **Supabase**
